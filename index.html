说明弹窗

release、archive使用cf加速，文件会跳转至JsDelivr

注意，不支持项目文件夹!

合法输入示例：

  • 分支源码 https://github.com/user/repo/archive/master.zip
  • release源码 https://github.com/user/repo/releases/download/v1.0/file.zip
  • release文件 https://github.com/hunshcn/project/releases/download/v0.1.0/example.zip
  • 分支文件 https://github.com/hunshcn/project/blob/master/filename

GitHub 文件加速

PS： GitHub文件链接带不带协议头都可以，支持release、archive以及文件，更多用法请参考这篇文章。
